## Paleta de colores

He elegido una paleta con colores oscuros y azules porque quería que el portfolio tuviera un estilo tecnológico, limpio y relacionado con la informática.

- Azul oscuro: `#0f172a`
- Azul claro: `#38bdf8`
- Blanco grisáceo: `#f8fafc`
- Gris texto: `#64748b`

## Tipografías

He usado dos fuentes de Google Fonts:

- Poppins para títulos.
- Inter para el texto normal.

## Elemento diferenciador

El elemento diferenciador de mi portfolio es un botón para cambiar entre modo oscuro y modo claro.

Lo elegí porque es una función sencilla pero útil, y además me permite practicar JavaScript básico. El botón cambia una clase en el `body` y, según esa clase, se modifican los colores definidos en el CSS.

## Uso de IA

He usado inteligencia artificial como apoyo para organizar la estructura del portfolio, mejorar algunos textos, revisar ideas de diseño y ayudarme con partes del código HTML, CSS y JavaScript.

Después he adaptado el contenido con mis datos reales, mis estudios, mis proyectos y mi forma de expresarme para que el portfolio sea personal y pueda entender lo que contiene.

## Dificultades encontradas

Una dificultad ha sido organizar bien todas las secciones para que la página quedara clara y no demasiado cargada.

También me ha costado elegir un diseño que fuera sencillo de entender pero que quedara visualmente cuidado. Para resolverlo usé una estructura de tarjetas, una paleta de pocos colores y un diseño responsive para que se vea bien en móvil y ordenador.

Otra dificultad fue añadir el modo claro y oscuro. Lo resolví usando JavaScript básico para añadir o quitar una clase al `body`.
