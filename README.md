# Portfolio personal - Víctor Miguel Jiménez Moreno

Este es mi portfolio personal web. Lo he hecho como trabajo de 1.º DAM para presentar mis datos, mi formación, mis habilidades y algunos proyectos que he realizado.

## Página publicada

https://victormjm1106.github.io/portfolio-victor-jimenez/

## Tecnologías usadas

- HTML
- CSS
- JavaScript básico
- GitHub
- GitHub Pages

## Colores usados

He elegido estos colores porque quería que la web tuviera un estilo oscuro y relacionado con la informática:

- Azul oscuro: `#0f172a`
- Azul claro: `#38bdf8`
- Blanco: `#f8fafc`
- Gris: `#64748b`

## Elemento diferenciador

El elemento diferenciador es un botón para cambiar entre modo oscuro y modo claro.

Lo elegí porque es algo sencillo, útil y me sirve para practicar un poco de JavaScript.

## Uso de IA

He usado IA como ayuda para organizar la página, mejorar algunos textos y resolver dudas de HTML, CSS y JavaScript.

Después he adaptado el contenido con mis datos reales, mis proyectos y mi forma de expresarme.

## Dificultades encontradas

Una dificultad fue organizar bien las secciones para que la página no quedara desordenada.

También me costó un poco hacer que la web se viera bien en móvil y ordenador. Lo solucioné usando CSS responsive y probando distintos tamaños de pantalla.

Otra parte que tuve que revisar fue el botón de modo claro y oscuro, porque necesitaba usar JavaScript para cambiar el estilo de la página.
